   - **Telefone:** +351 963 988 807 (ou o que tiverem)
   - **Website:** https://247barbearia.pt
   - **Horários:**

## ✅ PARTE 2: GOOGLE SEARCH CONSOLE (10 min)

### Passo 1: Adicionar Website

1. Abre: https://search.google.com/search-console
2. Clica **"Add Property"** (botão azul)
3. Seleciona **"URL prefix"**
4. Digita: `https://247barbearia.pt`
5. Clica **"Continue"**

### Passo 2: Verificar Ownership

1. Google pede para **verificar ownership**
2. Seleciona **"Meta tag"** (mais fácil)
3. Copia o código (algo como: `<meta name="google-site-verification" ...>`)
4. Entra em contacto com o programador para adicionar ao código
   - OU espera que ele envie confirmação de que já está adicionado
5. Volta ao Search Console e clica **"Verify"**

### Passo 3: Submete Sitemap

1. No Search Console, no menu esquerdo, encontra **"Sitemaps"**
2. Adiciona: `https://247barbearia.pt/sitemap.xml`
3. Clica **"Submit"**

### Passo 4: Verifica Coverage

1. Clica em **"Coverage"** no menu esquerdo
2. Confirma que estão indexadas:
   - / (homepage)
   - /index.html
   - Outras páginas (se existirem)

### ✅ VERIFICAR

- [ ] Site verificado (✅ verde)
- [ ] Sitemap submeto
- [ ] URLs indexadas

## ✅ PARTE 3: GOOGLE ANALYTICS (5 min)

### Passo 1: Criar Conta

1. Abre: https://analytics.google.com
2. Clica **"Start measuring"**
3. Preenche:
   - **Account name:** 24.7 Barbearia
   - **Website name:** 247barbearia.pt
   - **Website URL:** https://247barbearia.pt
   - **Industry category:** Beauty & Fitness (ou Similar)
   - **Reporting timezone:** Europe/Lisbon

### Passo 2: Copiar Código

1. Depois de criar, Google da um **código de rastreamento**
2. Envia esse código para o programador para adicionar ao website
3. Aguarda confirmação

### ✅ VERIFICAR

- [ ] Tag instalada no site (pode ver em DevTools → Network)
- [ ] Primeiros eventos aparecem em Analytics após ~24h
