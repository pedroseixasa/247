🖼️ URLs das Imagens dos Serviços

Copia estas URLs e adiciona no Admin Panel para ativar o hover nas imagens

📋 Como usar:

  1. Clica em "Copiar URL" abaixo de cada imagem
  2. Vai ao Admin Panel → Gestão de Serviços
  3. Edita o serviço correspondente
  4. Cola a URL no campo "URL da Imagem"
  5. Guarda
  6. Testa o hover no site! 🎉
